# E-commerce Repurchase Prediction System

Machine Learning application developed with Streamlit and XGBoost to predict customer repurchase probability.

## Technologies
- Python
- Streamlit
- Scikit-Learn
- XGBoost
- Pandas

## Features
- Interactive customer sliders
- Real-time probability prediction
- Classification of repurchase likelihood

## Model
XGBoost classifier trained on e-commerce behavioral variables.

## Deployment
Streamlit Cloud
